Project Overview

This project is a Basic to Advanced Calculator offering four main modes: Standard Calculator, Scientific Calculator, Programmer Calculator, and Unit Converter. It includes advanced functions such as trigonometry, logarithmic, exponential, and factorial calculations. The GUI is developed using Python's Tkinter and CustomTkinter, providing a clean and interactive interface for various calculations and unit conversions. The application is designed to be user-friendly and intuitive for seamless usage.

Features :
*Standard Calculator: +, -, ×, ÷, %, ±, √, x², 1/x, etc.
*Scientific Calculator: sin, cos, tan, sec, csc, cot, log, ln, exp, xʸ, 10ˣ, etc.
*Programmer Calculator: DEC, BIN, OCT, HEX, AND, OR, XOR, NOT, bitwise shifts.
*Unit Converter: Length, Weight, Temperature, Volume, Speed, Time.
*History & Last Answer (ANS) functionality.
*Clear and intuitive GUI interface.
*Responsive hamburger side menu to switch between modes.

Installation
Install Python 3.10+

Install required libraries:
pip install customtkinter

Run the project file:
python CALCULATOR.py

Usage
*Use the hamburger menu to switch between modes.
*Perform calculations using the calculator buttons.
*In the Unit Converter, select any category to convert values.
*Access the last calculation and result using the History and ANS button.
